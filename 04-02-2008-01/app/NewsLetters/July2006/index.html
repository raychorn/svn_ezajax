New Features Coming Soon !

ezAJAX™ Version 0.93 to be released soon, will contain a slick Tabbed Interface that is now being used by our home page. Our Tabbed Interface allows for one or many content pages to be associated with each Tab. This allows the programmer to define links within content pages that can link directly to a content page or a tab. This allows a book with chapters to be created using our Tabbed Interface. The content page that contains the index to the book would have links to content pages; the tabs would allow navigation to chapter headings. Navigation from content page to content page would allow the site visitor to turn each content page or jump between content pages.

The ezAJAX™ Design Philosophy (contin.)

ezAJAX™ uses a unique proprietary technique that allows ColdFusion Query Objects to be quickly transferred from the Server to the JavaScript client. This allows Query of Queries to be performed using the client to minimize having to go back to the Server just to run Queries. Query Objects can also be cached in the client and used at a later time whenever data is needed by the client. JSON does not typically support the ability to perform Queries on Data Objects once those data objects reach the client.

The ezAJAX™ way of transferring data from the server to the client allows the data objects to be used immediately with no need to perform any kind of JSON-like data transformations - no need to parse the data like XML does - no need to spend time decoding the data. This makes AJAX Clients work much faster when using ezAJAX™.
