The ezAJAX™ Design Philosophy (contin.)

ezAJAX™ uses a unique proprietary technique that allows ColdFusion Query Objects to be quickly transferred from the Server to the browser client using JSON. This allows Query of Queries to be performed using the client to minimize having to go back to the Server just to run Queries on Result Sets that were previously fetched from your database. Query Objects can also be cached in the client and used at a later time whenever data is needed by the client. We have extended JSON to support our own proprietary client-side Query Objects that support client-side Query of Queries.

The ezAJAX™ way of transferring data from the server to the client allows the data objects to be used immediately at the speed of thought. This makes your AJAX Clients work much faster when using ezAJAX™.
